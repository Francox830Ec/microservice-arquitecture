This project was built on microservices with hexagonal architecture, SOLID principles, DDD (Domain Driven Design), Event Driven Arquitecture with Apache KAFKA, reactive programming, CQRS and CDC (Change Data Capture) design patterns.

A relational database is used for writing and a noSQL database for reading data. In addition, a postman collection file has been added to test the different functional endpoints. Containers such as Docker are also used for the implementation of the main project as well as each of the microservices.

With these characteristics, the solution considers factors such as performance, scalability and resilience.
